# CAF Publication: Target Journals

**Last Updated**: November 10, 2025

---

## Primary Target

### 1. Bioinformatics (Oxford University Press)

**Impact Factor**: ~7.0 (2023)
**Scope**: Computational methods for biological data
**Acceptance Rate**: ~25%
**Time to Publication**: 4-6 months

**Article Type**: Application Note
- **Length**: 2,000 words max
- **Figures**: 3-4
- **Tables**: 2-3
- **Supplementary**: Unlimited

**Fit for CAF**:
- ✅ Novel computational method
- ✅ Software implementation required
- ✅ Performance benchmarks expected
- ✅ Open-source availability required
- ✅ High-impact journal in bioinformatics

**Submission requirements**:
- Manuscript in Word or LaTeX
- Figures in high-resolution (300+ dpi)
- Code availability statement
- Data availability statement
- Author contributions (CRediT)

**Pros**:
- Prestigious journal
- Wide readership in bioinformatics
- Application Note format fits CAF scope
- Strong computational biology focus

**Cons**:
- Competitive (high rejection rate)
- Strict word limits
- May prefer clinical applications

**Recommendation**: **Primary target** - Perfect fit for CAF

## Secondary Targets

### 2. BMC Bioinformatics

**Impact Factor**: ~3.0 (2023)
**Scope**: Computational biology and bioinformatics
**Acceptance Rate**: ~40%
**Time to Publication**: 3-5 months

**Article Type**: Software Article
- **Length**: No strict limit (~4,000 words typical)
- **Figures**: Flexible
- **Open Access**: Required (APC: ~$2,000)

**Fit for CAF**:
- ✅ Open access aligns with open-source
- ✅ Software focus
- ✅ More flexible format
- ✅ Reproducibility emphasis

**Pros**:
- Open access (wider visibility)
- More flexible format (not word-limited)
- Welcoming to software papers
- Faster review (~2-3 months)

**Cons**:
- Lower impact factor
- Article processing charge (APC)
- Less prestigious

**Recommendation**: **Strong backup** if Bioinformatics rejects

---

### 3. GigaScience (Oxford University Press)

**Impact Factor**: ~7.0 (2023)
**Scope**: Big data, reproducibility, computational biology
**Acceptance Rate**: ~35%
**Time to Publication**: 3-4 months

**Article Type**: Technical Note
- **Length**: Flexible (~3,000 words)
- **Focus**: Reproducibility and open data
- **Open Access**: Required (APC: ~$1,500)

**Fit for CAF**:
- ✅ Big data focus (alignment files)
- ✅ Reproducibility emphasis (perfect for CAF)
- ✅ Open science philosophy
- ✅ Data availability required (public benchmarks)

**Pros**:
- High impact factor
- Reproducibility focus aligns with research approach
- Open data mandate (strength for CAF)
- Flexible format

**Cons**:
- APC cost
- Less traditional bioinformatics audience
- Emphasis on data may overshadow methods

**Recommendation**: **Alternative** if want open access + high IF

## Pre-print Strategy

### bioRxiv

**Purpose**:
- Early visibility
- Establish priority
- Get community feedback

**Timing**:
- Submit to bioRxiv **same day** as journal submission
- Update after peer review

**Benefits**:
- Free
- Immediate visibility
- Can share before publication
- Indexed by PubMed

**Recommendation**: **Definitely use** bioRxiv

## Decision Tree

```
1. Submit to Bioinformatics (Application Note)
   ├─ Accepted → Great!
   ├─ Revise & Resubmit → Address reviews, resubmit
   └─ Rejected →
       ├─ If "out of scope" → Submit to GigaScience
       └─ If "quality concerns" → Address, submit to BMC Bioinformatics

2. If Bioinformatics rejects without review:
   → Submit to BMC Bioinformatics (Software Article)

3. If BMC Bioinformatics rejects:
   → Submit to GigaScience (Technical Note)

4. If all reject:
   → PLOS Computational Biology
   → Keep bioRxiv version as archival record
```
